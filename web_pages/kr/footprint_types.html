Footprint types

Through-Hole (TH) Footprint

설명 : PCB를 관통하는 구멍(홀)을 통해 부품의 핀이 삽입되고 납땜되는 방식.

특징 : 견고한 연결을 제공. DIP, TO, SIP 패키지에서 주로 사용. 수작업 또는 자동화된 삽입 공정에서 사용됨.

예시 : 저항, 커패시터(리드 타입), 트랜지스터, 릴레이.

Surface-Mount Technology (SMT) Footprint

설명 : PCB 표면에 직접 부품을 장착하며, 납땜 패드가 부품 핀 또는 단자와 맞닿음.

특징 : 부품 크기와 PCB 크기를 최소화 가능. 대량 자동화 생산에 적합. 고주파 특성이 우수함.

종류 :
SOIC : 소형 IC에 사용되는 평평한 패드.
QFP : 네 면에 핀이 배열된 IC용.
BGA : PCB 아래쪽에 볼 형태의 접촉점을 가진 패드.
QFN : 노출된 히트싱크와 패드를 가진 소형 고밀도 패키지용.

Chip Footprint

설명 : 칩 형태의 부품에 사용되는 소형 납땜 패드.

특징 : 소형 부품(칩 저항, 커패시터)에 적합. 일반적으로 직사각형 또는 정사각형의 단순한 패드 형태.

예시 : 0603, 0805, 1206 크기 표준으로 분류.

Connector Footprint

설명 : 전원, 데이터 또는 신호를 연결하는 커넥터용 Footprint.

특징 : 핀 간격과 핀 배열이 중요. 다양한 커넥터 표준에 따라 설계.

예시 : USB, HDMI, RJ45, Molex 커넥터.

Test Point Footprint

설명 : PCB 테스트 및 디버깅을 위해 신호 측정용으로 설계된 패드.

특징 : 표면 패드 또는 Through-Hole 형태로 제공. 프로브로 쉽게 측정 가능하도록 배치.

용도 : 제품 검사, 회로 디버깅.

Mechanical Footprint

설명 : PCB의 기계적 요소를 위한 구조적 디자인.

특징 : 나사 구멍, 스탠드오프, 히트싱크 부착 부분 등. PCB 조립과 외부 하우징 설계에 필수적.

Mixed Technology Footprint

설명 : Through-Hole과 SMT 방식이 혼합된 Footprint.

특징 : 동일 부품에서 두 가지 기술을 모두 활용.

예 : 파워 모듈, 커넥터 등이 PCB 표면에 부착되면서 동시에 핀이 관통.

Custom Footprint

설명 : 표준 부품에 맞지 않는 특수 부품이나 맞춤 설계 부품에 사용.

특징: 사용자 정의 규격에 따라 설계. 특수 목적 기기에서 사용.

Footprint 설계 시 고려사항

부품 크기 : 부품의 실제 크기와 핀 배열을 정확히 측정.
패드 크기 및 간격 : 납땜 및 신뢰성에 영향을 미침.
열 방출 : 열 관리를 위한 히트싱크 패드 포함 여부.
테스트 및 조립 편의성 : 충분한 공간 확보.
제조 표준 : IPC-7351 표준 준수 여부 확인.
