Embedded system designer J.H. Park

패키징 종류

DIP (Dual Inline Package, 듀얼 인라인 패키지)

설명 : 직사각형 형태로, 양쪽에 핀이 나란히 배열된 패키지.

특징 : 주로 손으로 조립이 가능한 PTH(Through-Hole) 기술에 사용. 핀이 길어 PCB에 꽂아 납땜하기 쉬움.

용도 : 소형 마이크로컨트롤러, 연산 증폭기 등.

SMD/SMT (Surface Mount Device/Technology, 표면 실장 기술)

설명 : PCB 표면에 부착되어 납땜되는 부품.

특징: 핀이 짧거나 없으며, 소형화 가능. 높은 제조 자동화율과 공간 절약.

SOIC (Small Outline IC) : 얇고 좁은 패키지.

QFP (Quad Flat Package) : 네 면에 핀이 배열된 평면 패키지.

BGA (Ball Grid Array) : 핀이 아니라 볼 형태로 접촉면이 구성된 패키지.

TO (Transistor Outline, 트랜지스터 아웃라인 패키지)

설명 : 주로 전력 트랜지스터나 다이오드에 사용되는 원형 패키지.

특징 : 열 방출 효율이 높음. 견고하고 다양한 크기로 제공.

용도 : 전력 증폭기, 다이오드 등.

SIP (Single Inline Package, 싱글 인라인 패키지)

설명 : 핀이 한쪽에만 배열된 패키지.

특징 : 간단한 구조로 제작. 상대적으로 부품 밀도가 낮음.

용도 : 저전력 회로, LED 어레이.

COB (Chip on Board, 칩 온 보드)

설명 : 칩을 PCB에 직접 부착하고 보호 코팅으로 덮는 방식.

특징 : 부품 크기를 최소화 가능. 제조 비용 절감.

용도 : LED 모듈, 센서, 디스플레이 패널.

QFN (Quad Flat No-Lead, 쿼드 플랫 노리드 패키지)

설명 : 핀이 보이지 않고 밑면에 접촉 패드가 있는 패키지.

특징 : 매우 얇고 소형. 열 방출이 우수.

용도 : RF 모듈, 소형 전자 기기.

CSP (Chip Scale Package, 칩 스케일 패키지)

설명 : 칩 크기에 가까운 크기의 패키지.

특징 : 부피가 작아 모바일 기기에서 유용. 고밀도 집적 가능.

용도 : 스마트폰, 메모리 칩.

TQFP (Thin Quad Flat Package, 씬 쿼드 플랫 패키지)

설명 : QFP의 얇은 버전으로, 핀이 네 면에 걸쳐 배열된 패키지.

특징 : 공간 절약 가능. 열 방출 성능 향상.

용도 : 마이크로컨트롤러, FPGA 등.

WLCSP (Wafer-Level Chip Scale Package)

설명: 웨이퍼 상태에서 직접 패키징하는 방식.

특징 : 가장 작은 형태의 패키지. 고밀도 전자기기에 적합.

용도 : 스마트폰, 웨어러블 디바이스.
